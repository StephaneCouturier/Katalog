Sélection

Résumé​

Cette page décrit toutes les fonctionnalités du panneau Sélection, la partie gauche de l'interface utilisateur.

  • Ce panneau permet d'affiner une sélection pour les différents écrans et fonctionnalités.
  • Avec le mode Recherche dans les catalogues de fichiers, la sélection filtrera les informations pour les écrans Rechercher, Périphériques, Créer et Statistiques.
  • Avec le mode Recherche dans les lecteurs connectés, un répertoire directement à partir des appareils connectés peut être sélectionné. Ceci n'est utilisé que pour l'écran Rechercher ;

Interface​

Boutons du haut :

  • Afficher/Masquer : Le bouton en haut à gauche peut être utilisé pour masquer et afficher à nouveau le panneau.
  • Réinitialiser : ce bouton avec l'icône en forme de balai réinitialise la sélection actuelle, de sorte que tous les catalogues/données soient sélectionnés.
  • Recharger : cela recharge l'intégralité des données de la collection, aidant par exemple à mettre à jour l'application pour les modifications apportées aux données en dehors de celle-ci.

Recherche dans les catalogues​

Informations de sélection : Cette section affiche la sélection actuelle de périphérique virtuel, de stockage ou de catalogue.

Le bouton à côté de l'étiquette de l'arborescence des appareils peut être utilisé pour faire une boucle entre l'affichage de l'arborescence entièrement réduite, le premier niveau ou le deuxième niveau.

Menu contextuel (clic droit)​

  • Recherche : avec sélectionnera simplement l'élément et accédera à l'écran de recherche
  • Explorer : cela ouvre le catalogue sélectionné dans l'écran Explorer
  • Mise à jour : cela déclenche la mise à jour (analyse des fichiers) du périphérique de stockage ou de catalogue sélectionné

Rechercher dans les lecteurs connectés​

Avec cette option, une recherche peut être effectuée directement dans n'importe quel répertoire de lecteur connecté/monté, sans avoir besoin d'un catalogue.
