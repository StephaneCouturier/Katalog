Explorer

Résumé​

Cette page décrit toutes les fonctionnalités de l'écran Explorer et comment les utiliser.

L'objectif de cet écran est d'explorer le contenu d'un catalogue de fichiers même lorsque le périphérique physique n'est pas connecté.

Cette vue chargera les données après un clic droit sur un catalogue à partir du panneau Sélection ou de la vue de liste Catalogues, et en choisissant l'entrée du menu contextuel Explorer.

Caractéristiques​

  • Afficher la liste des répertoires et fichiers du catalogue sélectionné.
  • Cliquer sur un fichier essaiera de l'ouvrir s'il est disponible
  • Un clic droit sur un fichier affichera le menu contextuel comme sur l'écran Recherche pour ouvrir le fichier ou son répertoire, ou copier des informations telles que le chemin du fichier.
  • Les dossiers peuvent être répertoriés avec des fichiers, ainsi que des sous-dossiers.
  • Si l'ordre de la liste est modifié, le bouton Trier les dossiers en premier listera d'abord les dossiers, par ordre alphabétique, puis les fichiers par ordre alphabétique.
